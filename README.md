# CETEP — Landing page

Landing page moderna e responsiva (HTML + CSS + JS) para o colégio técnico **CETEP**, inspirada no padrão institucional do **gov.br** (referência visual do Bolsa Família).

## Como abrir

- Abra o arquivo `index.html` no navegador.
- Os recursos (imagens/vídeos) são carregados de `./assets/`.

## Onde editar

- **Conteúdo**: `index.html`
- **Estilos**: `styles.css`
- **Interações** (menu mobile, animações, “voltar ao topo”): `script.js`

## Observações

- Os cursos listados em “Cursos oferecidos” estão como **exemplo** — ajuste para refletir a oferta real do CETEP.
